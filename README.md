## Installing Node.js

```
sudo apt update && sudo apt upgrade
```

```
wget https://raw.githubusercontent.com/nvm-sh/nvm/master/install.sh
bash install.sh
```

```
source ~/.bashrc
```

```

0.39.1
```

```

```
nvm use 18
```

```
nvm alias default 18
